# Laptop Onboarding

New laptops are prepared with standard security and productivity tools.

1. Connect the laptop to power and a trusted network.
2. Sign in with your company email address.
3. Complete MFA when prompted.
4. Allow device setup to finish before installing additional software.
5. Open the software portal to install role-specific applications.
6. Restart the laptop after the first setup cycle completes.

Required tools include endpoint protection, disk encryption, browser security extensions, office productivity apps, and the company VPN client. Contact IT if setup remains on the same screen for more than 30 minutes.
