# Printer Troubleshooting

If a printer will not print, check the basics before opening a ticket.

1. Confirm the printer is powered on and shows no paper jam or toner error.
2. Make sure your laptop is connected to the office network or VPN.
3. Remove stuck jobs from the print queue.
4. Restart the printer and wait two minutes.
5. Re-add the printer from the company printer directory if it still does not appear.

For secure print devices, make sure you released the job at the printer using your badge. Report repeated failures to IT with the printer name, location, and any error code on the display.
